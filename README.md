# HerdWell - Smart Livestock Monitoring System

HerdWell هو مشروع لمراقبة الماشية بشكل ذكي باستخدام **InfluxDB Cloud** و**Grafana**. 
يسمح بجمع بيانات مثل: درجة الحرارة، الرطوبة، مستوى الماء، ونشاط الحيوان، وعرضها على داشبورد تفاعلي.

## متطلبات المشروع
- Python 3.8+
- مكتبة influxdb-client
- حساب InfluxDB Cloud
- Grafana Cloud
